🎲 Randomnius Wiki

A fun, AI-powered Discord bot that keeps chats alive with topics, debates, and spicy takes 🔥

✨ Features

  • 💬 Auto Chat Reviver Sends topics when chat becomes quiet
  • 🧠 AI-Powered Content Uses Gemini AI to generate smart, fun, and clean prompts
  • 🎯 Category-Based Topics Generate topics based on categories like gaming, programming, etc.
  • ⚙️ Channel Control Enable/disable bot per channel
  • ⏱ Custom Delay System Set how long before the bot revives chat
  • 🔒 Admin Controls Staff-only commands for full control

📜 Commands

💬 Public Commands

/randtopic
Generate a random topic Optional: category (gaming, programming, funny)

/wouldyourather
Generates a "Would You Rather" question

/hottake
Generates a controversial hot take 🔥

/debate
Generates a debate topic ⚔️

🛠 Staff Commands (Admin Only)

/settopicchannel
✅ Allows the bot in the current channel
Perk: Enables all topic commands in that channel

/deltopicchannel
❌ Disables the bot in the current channel
Perk: Stops all bot activity there

/setquietdelay <minutes>
⏱ Sets inactivity time before bot sends a topic
Perk: Customize chat revival timing

/togglequiet
🔄 Turns auto-topic system ON/OFF
Perk: Full control over automatic messages
