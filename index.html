Randomnius - Open Source

Keep your Discord server chat alive with randomly-generated topics, debates, and fun questions!

Invite To Your Server Documentation

Features

  • 💬 /randtopic – Random discussion topics (gaming, programming, funny)
  • 🤔 /wouldyourather – Fun “Would You Rather” questions
  • 🔥 /hottake – Spicy hot takes for debate
  • ⚔ /debate – Intresting debate topics
  • 🕒 Dead-chat reviver – Posts topics automatically when chat is quiet

Commands

  • /randtopic [category]
  • /wouldyourather
  • /hottake
  • /debate

Staff Commands

Used By Users With "Administrator" Permission via roles or permissions

  • ⏳ /setquietdelay – Set inactivity reminder delay(in minutes)
  • 🤔 /togglequiet – Toggles chat inactivity Reminder
  • 🔥 /settopicchannel – Allows Randomnius on the working channel
  • ⚔ /deltopicchannel – Disables Randomnius on the working channel
